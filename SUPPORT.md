## What to include when asking for help

Please send:

- your issue in one or two clear sentences,
- your Windows version,
- the Memory Palace version if visible,
- whether the issue happens during ingest, compile, chat, or output generation,
- a screenshot or screen recording if the issue is visual,
- a sample source file only if you are comfortable sharing it.

## What this repo is for

This repository is for:

- product information,
- public FAQ,
- roadmap visibility,
- support routing.

This repository is not the app source repository.

### Bug reports

If GitHub Issues are enabled, use an issue for non-sensitive bugs.

Do not post:

- API keys,
- private documents,
- screenshots with confidential data,
- license keys,
- machine-specific secrets.

For anything sensitive, use email instead: `alphatech.digitolead@gmail.com`.
